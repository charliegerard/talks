What does it work?

Johnny-five

Open-source JavaScript framework base on the Firmata protocol.

Firmata is a protocol for communicating with microcontrollers from software on a host computer.

Getting started / setup

Prerequisites (OSX)

  • Install Node
  • Install Xcode
  • Install node-gyp

Getting started / setup

Prerequisites (Window)

  • Install Node.js >= 0.10.x 32 bit
  • Install Visual Studio community edition
  • Install Python
  • Open up cmd (Start > Run.. > cmd) and enter set PATH=%PATH%;C:\Python27
  • Install node-gyp

"Hello World" example

``` var five = require("johnny-five"); var board = new five.Board(); board.on("ready", function() { // Create an Led on pin 13 var led = new five.Led(13); // Blink every half second led.blink(500); }); ```
